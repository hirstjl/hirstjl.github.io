To convert Maple animations to Quicktime movies (with sliders)

  1. Enter and execute the animation command in Maple.
  2. Resize the animation window as desired. (It's better to do this now.)
  3. With the animation window selected, go to the File menu and select "Export as html." Choose GIF output, and note where the GIF will be saved.
  4. Wait. You have time for coffee.
  5. Maple will export an html page displaying your animation. The animation is stored in animated GIF format. Now we'll convert the animated GIF to a Quicktime movie, so that we get that nice slider.
  6. Start Quicktime Pro. Under the file menu, select "Open movie in new player," and then select the GIF. You can either "save-as" a self-contained movie, or "export" as a movie. If you export, selecting options-settings-graphics will reduce the file size substantially.

This process was used to create the animations on the main page.
