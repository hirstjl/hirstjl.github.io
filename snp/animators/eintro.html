A Graphical Tool for Linear Algebra

The procedure eshow graphically illustrates some concepts from introductory linear algebra. When passed a 2x2 matrix A as an argument, the procedure computes the product Au for each of several unit vectors u, and displays the results as an animation. Additionally, each frame contains textual annotations including decimal approximations of u, |u|, Au, and |Au|.

You can view a demonstration animation generated by eshow using your browser. (To create new animations for matrices of your choice, you will need access to Maple.)

To use eshow, follow these steps...

    If you have installed Maple V Release 4 as a helper application on your browser, you can load the eshow worksheet.

    Otherwise, following these instructions:

    • Cut the Maple commands from the Maple for eshow page and paste it into a Maple V Release 4 worksheet. Press enter.
    • Enter a matrix. For example, use the command A:= matrix([[2,0],[1,1]]); Press enter.
    • Enter the command eshow(A); Press enter, and watch your animation.
