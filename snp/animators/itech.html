Technical Notes and Pedagogy

Technical Notes:

  • Maple V release 3 is sufficient to execute these commands.
  • Animations can be generated more quickly by reducing the number of frames.

Pedagogy:

Students should be encouraged to note that the parametric equations of the involute of the circle are used in the second animation. Good student projects include modifying the code to draw "string and pencil" constructions of other figures, especially conic sections.
