Generating the Involute of a Circle

This page discusses Maple code that generates two animations. One is an animation of string unwinding from a circle. The second is an animation of the string, including the path traced by the end of the string. The added curve in the second animation is called the involute of the circle.

You can view a demonstration animation using your browser. (To modify the code and produce new animations, you will need access to Maple.)

To use these commands...

    If you have installed Maple V Release 4 as a helper application on your browser, you can load the involute worksheet.

    Otherwise, following these instructions:

    • Cut the Maple commands from the Maple for involute page and paste it into a Maple worksheet. Keep pressing enter.
