Precalculus with Online Homework



In Fall 1999, Jeff Hirst taught two sections of Precalculus using online homework assignments.

  • One assignment was given each week.

  • Each assignment could be repeated up to five times.

  • The problems varied with each repetition.

  • The highest grade was recorded.

The homework assignments were created using WebCT, a commercial set of Web-based Course development Tools.
